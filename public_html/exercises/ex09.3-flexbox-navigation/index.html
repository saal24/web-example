Instructions

Your task is to create an HTML document which has a navigation bar on the top. Use the conventional way with semantic elements: <nav> for the main navigation container, <ul> inside it and each menu item should be a list item (<li>).

Hints:

  • You probably want to style the <li> as a flexbox?
  • An NTNU logo image is inside the img directory.
  • If you do it properly, the navigation bar will fill the window width, the logo will be on the left and the menu items (the texts) will be on the right.
  • If you have a hard time removing whitespace on top of the navigation, you can set margin to zero for all elements with this CSS hack: * { margin: 0; }
