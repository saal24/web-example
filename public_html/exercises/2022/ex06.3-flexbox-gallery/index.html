Go back to exercise list.

Purpose

The purpose of the exercise is to practice configuring container properties for CSS flexbox. You may need to set some properties for the images, but no flexbox-item properties.

Instructions

Use this document as a template. You see a set of images below. Configure the following behaviour for the flexbox container:

  1. The images should be in a flexbox container.
  2. Each image should be 200px wide and 150px high.
  3. The container should fill the whole width of the page, with 50px margins on the sides.
  4. The container should not shrink to less than 500px horizontally (hint: min-width property).
  5. The container should be 800px high.
  6. The images should be laid out horizontally and have equal spacing between them and on the sides.
  7. If there is not enough space to lay out the images in a single row, the gallery should have an "automatic wrapping" - the images should be wrapped to a new line.
  8. The images should be centered vertically within the container. Add some border to the container to see that they are centered.
  9. The images should have a minimum margin of 20px. I.e., there could be more space between each two the images, but no less than 2*20px.

The following CSS properties may be handy (for the container):


        display: flex;
        justify-content
        align-content
        flex-wrap
