Exercise 6.1

This is Exercise #6.1, lecture on CSS Flexbox, used in the course IDATA2301 Web technologies at NTNU, campus Aalesund.

Go back to exercise list.

Instructions

Your task is to create an HTML document and necessary CSS styling to get the result described below. You decide whether to embed <style> block in the heading, or to include an external .css file.

Requirements:

  1. Create a flexbox container.
  2. It must contain five child elements, let's call them items.
  3. Each item should contain some text in it. For example, numbers 1, 2, ..., 5.
  4. Add a border to the flexbox container and each item to see their boundaries.
