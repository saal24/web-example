Resetting REM

In this example we reset rem to 10px. This is a typical technique for web developers - makes the usage of rem units easier. How do we do that? We redefine the font size for html tag. Because that is rem by definition: 1 rem = font size of html element. Check out the <style> section in this document!

After this manipulation, text becomes tiny. But we can easily fix that by setting font-size to necessary elements.
