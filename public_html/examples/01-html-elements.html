Primitive HTML elements

This site showcases primitive HTML elements which are used to construct components.

Text

There are different possibilities to display text - paragraphs, headings. In general, only the semantics (meaning) of the different text blocks differs. Styling is the same for paragraphs, headings and other text types. We can emphasize some parts of the text in HTML. But that should be used only when it is semantically necessary, not simply to get an italic text.
