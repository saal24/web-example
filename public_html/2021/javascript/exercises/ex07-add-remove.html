Exercise 7

This is Exercise #7, lecture on modifying DOM with Javascript, used in the course IDATA2301 Web technologies at NTNU, campus Aalesund.

Go back to exercise list.

Purpose

The purpose of the exercise is learn how you can remove elements from the DOM and add new ones.

Instructions

Part 1 - add a paragraph on button-click

Create a document with a button. Create Javascript which adds a <p>Hello!</p> to the document every time the user presses the button.

Part 2 - add a table row

  1. Extend the previous example: create a table with two columns. Add one row in it, with any content you want. Add a button. Every time the user clicks on the button, add a new row to the table with text "Hello" in one column and "World" in the other column. Hint: <table> has <tbody> as the direct child, rows should be added to the <tbody> element not to the <table> directly!
  2. Extend the previous example: add a button "Remove last row". When the user clicks on it, the last row of the table is removed. If the table has only one row left, an alert is shown saying "Can't remove the last row", and the row is not removed.

Part 3 - div inside a box

Add a <div> element to the page, set it's height and width to 100px, background color to green. Make the <div> element escape - when the user moves the mouse over it, the <div> element is removed from the place it currently sits and is moved to a random place in the table above. I.e, it is added to a random row and random column inside the table you created in the previous steps.

Part 4 - div outside the box

Extra challenge: add some more content to the page (some headings, paragraphs, etc) and make the div escape randomly at any place in the DOM tree. I.e., don't limit it's placement to the table.
