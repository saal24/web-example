Exercise 1

This is Exercise #1, lecture on CSS Flexbox, used in the course IDATA2301 Web technologies at NTNU, campus Aalesund.

Go back to exercise list.

Purpose

The purpose of the exercise is to get your hands dirty with CSS flexbox layout. Well, and to get used to writing HTML and CSS code. It's like playing piano - practice is needed to get it in your fingers.

Instructions

Your task is to create an HTML document and necessary CSS styling to get the necessary result described below. You decide whether to embed <style> block in the heading, or to include an external .css file.

Requirements:

  1. Create a flexbox container.
  2. It must contain five child elements, let's call them items.
  3. Each item should contain some text in it. For example, numbers 1, 2, ..., 5.
  4. Add a border to the flexbox container and each item to see their boundaries.

Side note(s)

P.S. If your code editor supports Emmet (WebStorm does), you can create the child items with this emmet-shortcut: div{$}*5. Type it and then press Tab (or Ctrl+E in some editors). Emmet's Cheat sheet here.

Explanation: div{$}*5 means "create element <div> with the loop index as the text content, repeat that in a loop 5 times".

A small quiz-question for you: how will this Emmet expression be expanded: div#item$*5? Hint: how about you try it and see for yourself? :)
