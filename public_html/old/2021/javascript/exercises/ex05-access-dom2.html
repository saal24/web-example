Exercise 5

This is Exercise #5, lecture on traversing DOM with Javascript, used in the course IDATA2301 Web technologies at NTNU, campus Aalesund.

Go back to exercise list.

Purpose

The purpose of the exercise is learn how you can select more than one element and traverse the children.

Instructions

Part 1 - inspect an existing site in the console

Go to https://web-tek.ninja in your browser (preferably in another tab), open developer tool console and explore the DOM:

  1. Select all elements with class css-exercise. You should get all 6 <li> elements containing links to CSS exercises. But you probably got only 5 items, right. Can you find what is missing? P.S. Debugging and finding mistakes in HTML, CSS and Javascript is an essential skill!
  2. Select the first <article> element on the page, store it in a variable.
  3. Now use that variable to find all <h2> elements inside that article. How many did you get? What do they refer to?

Part 2 - Create your own page

  1. Create an HTML document with a 3x3 grid container. An opportunity to remember how CSS grids work ;).
  2. Put 9 div elements inside it.
  3. For even div elements add class="even" (div number 2, 4, …, 8), for odd elements (div number 1, 3, …, 9) add class="odd".
  4. Write a Javascript code (execute it when the page is loaded) which sets text to "X" for all odd elements (those with class="odd") and sets text to "O" for all elements with class="even".
  5. Extra challenge: find out how you can do the same even if you don't assign any classes to the nine divs. (Find out how can you access odd and even child elements separately in Javascript)
