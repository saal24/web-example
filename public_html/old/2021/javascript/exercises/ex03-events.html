Exercise 3

This is Exercise #3, lecture on Javascript basics, used in the course IDATA2301 Web technologies at NTNU, campus Aalesund.

Go back to exercise list.

Purpose

The purpose of the exercise is get familiar with handling events in Javascript: executing a code when the whole document is loaded, reacting on a button click and text change in an input field. You will also practice using timers.

Instructions

Create a new HTML document, include a Javascript block there.

  1. Create a function which prints the square root of 1764 in the console. Call this function when the document is loaded.
  2. Print "Hello, i" (where i is a counter, incremented by one every time) to the console every second.
  3. Create a page with two buttons A and B. Print a message to the console (a different message for each button) whenever a button is pressed.
  4. Create a page with a text input field. Whenever the text changes there, print a message in the console: "The text changed".
